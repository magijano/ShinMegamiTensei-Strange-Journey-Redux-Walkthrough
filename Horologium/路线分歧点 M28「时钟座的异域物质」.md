# 戈尔Boss战<br>
- Boss Lv65 「超人戈尔」
  弱<font color = "green">风</font> 反<font color = "orange">物枪</font>
  所有技能都是<font color = "orange">物理</font>或<font color = "orange">枪</font>属性，携带上对应耐性的仲魔，多多使用「物反盾」即可轻松过关
  使用「蓄力」后一定会对主人公进行攻击，请注意防御
  死亡时会满血复活一次，之后一定会对主人公进行攻击，请注意防御
